## Getting Started

### Prerequisites

Yarn 0.25+

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/maxam2017/hotpink.git
   ```
1. Install dependencies
   ```sh
   yarn install
   ```
1. Run parcel-bundler
   ```sh
   yarn dev
   ```
